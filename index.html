Simulador de nómina para alumnos, con autocorrección y guardado en el navegador

Esta versión está pensada para subirla a internet fácilmente. No necesita base de datos. Funciona muy bien para clase, prácticas, tareas y autoevaluación.

Qué tiene esta versión

  • Modo alumno y modo profesor.
  • Autocorrección de ejercicios.
  • Guardado local en el navegador.
  • Historial de nóminas guardadas.
  • Impresión o PDF.
  • Un solo archivo fácil de subir.
Bueno para ti ahora: se puede subir a GitHub Pages o Netlify sin tocar servidores.
Límite importante: como no hay base de datos, cada alumno guarda sus datos solo en su propio navegador.

Pasos para publicarla

  1. Crear archivo index.html.
  2. Pegar este código.
  3. Subirlo a GitHub Pages.
  4. Compartir el link con tus alumnos.

Panel principal
